# Gentum.ar - App Móvil

## Descripción

Aplicación móvil de Gentum.ar construida con React Native y Expo.

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar Expo
npm start

# Ejecutar en iOS
npm run ios

# Ejecutar en Android
npm run android

# Ejecutar en web
npm run web
```

## Estructura

- `App.tsx`: Componente principal con navegación
- Pantallas principales: Inicio, Experiencias, Grupos, Comunidad, Perfil
